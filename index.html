    _col

    @function _col($col, $con: null) { ... }

    Description

    Возвращает ширину колонок

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $col

    Количество колонок для которых необходимо вернуть ширину в процентах

    		Number— none
    $con

    Контейнер для которого нужно вернуть ширину колонок в пикселях (не обязателен)

    		Null or Stringnull

    Returns

    Number —

    Возвращает ширину колонки в процентах если второй параметр не указан, и в пикселях если - указан

    Example

    // Для примера взято общее кол-во колонок - 12 и ширина md-контейнера 960px
_col(5) // 41.66667%
_col(5, md) // 400px

// 		font-family: "Raleway";
// 		src: url("/fonts/Raleway-Regular/Raleway-Regular.eot?#iefix") format("embedded-opentype"), url("/fonts/Raleway-Regular/Raleway-Regular.woff") format("woff"), url("/fonts/Raleway-Regular/Raleway-Regular.ttf") format("truetype");
// 		font-weight: 400;

    Description

    Разворачивание параметров для создания блока с фиксированным соотношением сторон

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value

// 		content: '';

// 		padding-bottom: 56.25%;

// 	}

// Вызов:
video
	+aspect-ratio(none)

// Вернёт:
//  video::before, video::after {
// 		display: none;
//  }
